Команды git

git add:
Добавляет новые или измененные файлы
git commit:
Записать изминения в репозиторий
git status:
Показать статус рабочего дерева
git pull:
Используется для извлечения и загрузки содержимого из удаленного репозитория в локальный
git fetch:
Загружает коммиты, файлы и ссылки из удаленного репозитория в ваш локальный репозиторий
git push:
Используется для выгрузки содержимого локального репозитория в удаленный репозиторий
git merge:
Выполняет слияние отдельных направлений разработки !?!
git stash:
Позволяет на время сдать в архив изменения, сделанные в рабочей копии, чтобы вы могли применить их позже
git checkout:
Переключение между различными версиями целевого объекта
git branch:
Позволяет создавать, перечислять, переименовывать и удалять ветки
